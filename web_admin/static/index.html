导入规则

总目录里的每个压缩包或子目录会成为独立语音包；例如“PDD .zip1”会显示为“PDD”。

文件名去掉扩展名后就是语音内容，只清理末尾的 p1 / p2 编号和“音频1”这类技术后缀。

重复文件自动跳过；试听会临时转 wav，发送仍使用原始 silk。
